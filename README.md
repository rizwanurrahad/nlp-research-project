# NLP Research Project (Repo-Ready)

This repo-ready package is prepared for a master's application section that asks for:
- one public repo link,
- core model code path,
- deployment-oriented detail,
- short role description.

## Suggested repository name
`nlp-research-project`

## Contents
- `src/model.py` - lightweight text classification pipeline
- `src/app.py` - FastAPI inference route
- `src/utils.py` - preprocessing helpers
- `requirements.txt` - dependencies
- `Dockerfile` - deployment-oriented detail for M2 applications
- `notebooks/nlp_research_demo.ipynb` - notebook version of the pipeline
- `APPLICATION_RESPONSE_TEMPLATE.txt` - copy-ready text for the application form

## How to use for your application
1. Create a GitHub repo named `nlp-research-project`.
2. Upload these files.
3. Make at least three commits.
4. Copy the real commit SHAs from GitHub.
5. Use the template in `APPLICATION_RESPONSE_TEMPLATE.txt`.

## Important honesty note
Do not invent commit SHAs. They only exist after real commits are made to Git.
